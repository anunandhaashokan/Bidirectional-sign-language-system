# Real-Time Bidirectional Sign Language Communication System

## Overview
The Real-Time Bidirectional Sign Language Communication System is an AI-powered accessibility solution designed to bridge the communication gap between hearing and speech-impaired individuals and people unfamiliar with sign language.

The system performs real-time translation of sign language gestures into text or speech and also supports reverse communication by converting spoken language or text into sign language representations. The project focuses on improving accessibility, inclusion, and communication efficiency in real-world environments.

---

## Objectives
- Enable real-time communication using sign language recognition
- Convert gestures into readable text and speech output
- Support speech-to-sign communication
- Improve accessibility in education, healthcare, and public services
- Provide an adaptive and user-friendly communication platform

---

- Real-time hand gesture detection

---

## Technologies Used
- Python
- OpenCV
- MediaPipe
- pyttsx3
- Computer Vision
- Deep Learning Concepts
- Gesture Recognition
- Speech Processing

---

## System Workflow
1. Capture live hand gestures using webcam
2. Detect hand landmarks using MediaPipe
3. Recognize gestures through gesture mapping and processing
4. Convert detected gestures into text and speech
5. Perform reverse conversion from speech/text to sign communication
